How good is your memory?.
Press Any Key to Start

Pay attention to the flashes of colors and remember the sequence. After the sequence is shown, repeat the sequence by clicking the buttons in the same order. If you enter the correct sequence, the game will continue with an additional step. If you make a mistake, the game will end and you can start over. Each level increases the difficulty by adding one more step to the sequence.
